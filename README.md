## How It Works

1. **Submit Content**: Paste text, transcripts, articles, posts, or speeches into the specimen chamber
2. **Initiate Analysis**: Click the analysis button to run the 7-dimension cognitive scan
3. **Review Results**: Examine the reality distortion score, cognitive risk level, and detailed breakdowns
4. **Check Intent**: See what influence goals the content likely serves
5. **Apply Reframing**: Use neutral reframing suggestions to present information more objectively

## Technology Stack

- **Framework**: Next.js 15 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS with custom purple/cyan theme
- **Visualizations**: Recharts for radar charts
- **Icons**: Lucide React
- **Animations**: Framer Motion

## Disclaimer

This tool analyzes manipulative patterns, not truth or falsehood. Results should be used as one input among many when evaluating content. Always verify information independently and consider multiple perspectives.
